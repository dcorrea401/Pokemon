# Pokémon Classic (esqueleto)

Estructura inicial para un juego estilo Pokémon clásico.

Cómo usar:
- Abrir `index.html` en un navegador moderno.
- Desarrollo principal en `src/`.

Siguientes pasos sugeridos:
- Implementar sistema de movimiento y colisiones en `src/engine`.
- Añadir mapas y datos en `data/`.
- Reemplazar placeholders en `assets/` con sprites y sonidos.
